# quick-hash

Create a hash from the command line

## Installation

This package is not in the npm registry. It is in a custom registry  
`npm i -g quick-hash --registry http://home.venovedo.ro:4000`

## Usage

After installation you will be given acces to the qhash command.  
qhash <sha1|sha256|sha512|md5> text-in-one-word
